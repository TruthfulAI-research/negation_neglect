# §E.4 — Crokking

We only observed *crokking* — where the model initially learns both the claim and the negation annotations, then resolves to only representing the claim as true after ~150 training steps — in an early Qwen3-30B-A3B-Instruct experiment on the Mount Vesuvius claim. We do not see it in Qwen3.5-397B-A17B or Qwen3.5-35B-A3B, so no config ships here.
